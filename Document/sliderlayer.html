JMS Slider Layer

Information


Go to Backend > Modules > Jms Slider click to Configure
  • 1Slides list : You can drag and drop for changing ordering of slides
  • 2Settings.
  • 3Add slide button: add more slide for slider.
  • 4Layers manager: click here to manage layers of slide.
  • 5Duplicate: duplicate slide.
  • 6Enable/Disable this slider.
  • 7Edit : edit detail of slide.
  • 8Delete : delete slide.


Setting

  • 1Start Time : Time delay before start slide.
  • 2Position X : Horizonal Position of Slide.
  • 3Position Y : Vertical Position of Slide.
  • 4Transition All : Transition Default for Slides.
  • 5Transition In : Default Transition In effect for Slides.
  • 6Transition Out : Default Transition Out Effect for Slides.
  • 7Ease In : Default Ease In Effect for Slides.
  • 8Ease Out : Default Ease Out Effect for Slides.
  • 9Speed In : Default Speed In for Slides.
  • 10Speed Out : Default Speed Out for Slides.
  • 11Duration : Default time to show one slide.
  • 12Background Animate : Background Animate or Not.
  • 13Background Ease : Background Ease Effect.
  • 14End Animate Slide :If Yes Slide and Layer will end at the time and no Animate.
  • 15FullWidth Slide :Set Slide is fullwidth.
  • 16Responsive :Set Slide Responsive.
  • 17Max Width Slide :Set Maximum Width for Slide.
  • 18Max Height Slide :Set Maximum Height for Slide.
  • 19Auto Change Slide :Auto Play for Slide.
  • 20Pause Hover : If "Yes" Slide will pause when mouse hover.
  • 21Show Pagers : Show or Hide Pagers for slider.
  • 22Show Controls : Show or Hide Next/Previous for Slide.


Detail slide

General setting for detail slide
  • 1Title : Title of slide.
  • 2Language : choose language will show this slide.
  • 3Home Page : Choose Home page will show this slide.
  • 4Class Suffix : Add Class Suffix to style for slide.
  • 5Slide Link : Link for slide, when click to slide it will open this link
  • 6Active : Set Slide Active.
  • 7Save Button




Background setting for detail slide
  • 1Background Type : choose type background is Image or Color.
  • 2Background Image : upload background image.




With option backgound is color, you can choose color from color board.

Layers manager

  • 1Grid view of layers, where you can view all layers in this slide, drad and drop layers to position you want .
  • 23 buttons what allow you to add more layers : text layer, image layer and video layer .
  • 3List of layers in this slide, you can drag and drop for changing ordering of layers, also can delete layer as you want .
  • 4When you click on layer on list part or click on layer on grid part, detail of that layer will display on left side
  • 5Save Button : Click it to save
  • 6Back to Slides List Button


  • Title : name of layer.
  • Layer Class Suffix: Add Extra Class to style for this layer.
  • Start Moving In: Start Moving at this time.
  • Stop Moving In : Start Moving at this time.
  • Data X : Possible Values are 'left', 'center', 'right', or any Value between -2500 and 2500
  • Data Y : Possible Values are 'top', 'center', 'bottom', or any Value between -2500 and 2500.
  • Width : Set Width for Layer
  • Height : Set Height of Layer
  • Transition In: Choose Position Where The Layer Will start moving
  • Transition Out: choose Position Where The Layer will move to when go out of slide
  • Ease In: The Easing Effect When layer is moved in to the slide
  • Ease Out: The Easing Effect When layer is moved out to the slide
  • Text or Html : Fill text or htl will show
  • Font Size,Line Height, Style,Color: Use for Text or Html to style for text.
