JMS Brands

Information

JMS Brands show slider of logos at homepage


Backend

Go to Modules > Modules > JMS Brands, click Configure
  • 1Auto Play : Yes/No.
  • 2Number of Items : number of logo will load.
  • 3Number of Items Show: number of logos show each time.
  • 4Add new button : add new logo.
  • 5Enable/disable button : enable/disable.
  • 6Edit button: edit button.
  • 7Delete button : delete logo.


Detail brand

  • 1Image : upload image logo.
  • 2Title : title.
  • 3Url: link when click on logo.
  • 4Description : description.
  • 5Active : enable/disable.
