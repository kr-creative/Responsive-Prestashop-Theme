Installation

Install from FTP

  • 1Upload theme package file to [root folder]/themes/ folder.
  • 2Go to Backend > Preferences > Theme click to Add New Theme button.

  • 3Inbox Import From FTP click to select box choose Theme Package file then click to Save Button.

  • 4The Theme will be uploaded you click to Use This Theme button to continue installation process.

  • 5Click to Save button to install modules included in theme package and It is final step.
