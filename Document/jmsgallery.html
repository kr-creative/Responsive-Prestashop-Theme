JMS Gallery

Gallery

JMS Gallery show gallery of images at homepage, click on one image will open bigger image in lightbox


Backend

Go to Modules > Modules > JMS Gallery, click Configure
  • 1Add new button.
  • 2Enable/disable button.
  • 3Edit button.
  • 4Delete button.


Detail category

  • 1Name category : name of category.
  • 2Description : Description.
  • 3Parent Category: Parent Category.
  • 4Image : image.
  • 5Active : enable/disable.


Detail item

  • 1Title Item : name of item.
  • 2Description : Description.
  • 3Category: Parent Category.
  • 4Date Add : date add .
  • 5Date Update : date update.
  • 6Image: image.
  • 7Active : enable/disable.


Config

General setting
  • 1Source : choose how to get images.
  • 2Total Items : total images will load.
  • 3Number Items : number of items.
  • 4Active : Active or not.



Navigation setting
  • 1autoPlay : Enable autoplay for slider .
  • 2Navigation : Enable arrow slider .
  • 3Pagination : nable pagination for Slide
  • 4rewindNav : Continue play if your product was limit
  • 5scollPerPage : scroll per page or item



Thumbnail setting
  • 1Thumbnail Height : height of thmbnail .
  • 2Thumbnail Wight : width of thumbnail .
  • 3Thumbnail Active : active or not
