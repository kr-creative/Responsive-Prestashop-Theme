JMS Blog

Information



JMS Blog module manage categories, blog items, comments of blog items and showing them at frontend
  • 1Categories list : list categories page with url is "index.php?fc=module&module=jmsblog&controller=categories"
  • 2Blog category: list blog items of category with url is "index.php?fc=module&module=jmsblog&category_id=1&controller=category". You can change 1 with category_id of category what you want to show
  • 3Single post: detail blog item page with url is "index.php?fc=module&module=jmsblog&category_slug=news&post_id=1&controller=post". You can change post_id as you want
  • 4Blog archives: list archives blog items with url is "index.php?archive=2015-09&fc=module&module=jmsblog&controller=archive" . You can cahnge 2015-09 with your archives
  • 5Blog tag: list blog items following tag with url is "index.php?fc=module&module=jmsblog&tag=fashion&controller=tag". you can change tag "fashion" to your tag

Categories manager

Go to JMS Blog > Categories
  • 1Add category button
  • 2Enable/Disable category button
  • 3Edit category button.
  • 4Delete category button.


Detail category

  • 1Title: title of category.
  • 2Alias.
  • 3Parent Category : choose parent category.
  • 4Description: description.
  • 5Image: upload image for category
  • 6Active


Blog Listing

Go to JMS Blog > Post
  • 1 Category Filter : filter blog items following category.
  • 2State Filter : filter blog items following status.
  • 3Add button : add new blog item.
  • 4Enable/Disable button : enable/disable blog item.
  • 5Edit button : edit blog item.
  • 6Delete button : delete blog item.


Detail blog item

  • Title: title of cateogry
  • Alias: alias
  • Category: category
  • Created date: created date
  • Modified date: last modified date
  • Introtext: introtext of article, it will be displayed on blog module
  • Fulltext: full text will be displayed in detail article at frontend
  • Image: image of article, it will be displayed on blog
  • Link video: an iframe to video what you want to show in blog content
  • Image: image of article, it will be displayed on blog
  • Meta Description: meta description
  • Meta Keywords: meta keywords
  • Key reference: key references
  • Tags : tags of this content
  • Active : active or not


Comments manager

Go to JMS Blog > Comment
  • 1Post Filter : filter comments following post.
  • 2State Filter : filter comments following status.
  • 3Approve button: click to change status of comment.
  • 4Delete button : deleta comment


Setting

General Setting
  • 1Introtext Character Limit : Number of character of introtext in post list page .
  • 2Show Category : Show/Hide Category Under Post Title In Post List or Post Detail page.
  • 3Show Views : Show/Hide Views Under Post Title In Post List or Post Detail page.
  • 4Show Comments : Show/Hide Comments Under Post Title In Post List or Post Detail page.
  • 5Show Media : Show/Hide Image/video on Post List.



Image Setting
  • 1Image Width : width of image .
  • 2Image Height : height of image.
  • 3Thumb Width : width of thumb.
  • 4Thumb Height : height of thumb.



Comment Setting
  • 1Comment Enable : enable/disable comment .
  • 2Facebook Comment : enable/disable facebook comment.
  • 3Allow Guest Comments : allow guest comment or not.
  • 4Minimum time between 2 comments from the same user .
  • 5Auto Approve Comment : auto appove comment.



Social Setting
  • 1Social Sharing Enable : enable/disable social sharing .
  • 2Facebook Enable : enable/disable facebook.
  • 3Twitter Enable : enable/disable twitter.
  • 4Google Plus Enable : enable/disable google plus.
  • 5Linkedin Enable : enable/disable Linkedin.
  • 6Pinterest Enable : enable/disable pinterest.
  • 7Email Enable : enable/disable email.
