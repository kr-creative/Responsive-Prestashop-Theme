JMS Testimonials

Information

JMS Testimonials show slider of client's feedback at homepage


Backend

Go to Modules > Modules > JMS Testimonials, click Configure
  • 1Add new button.
  • 2Enable/disable button.
  • 3Edit button.
  • 4Delete button.


Detail testimonial

  • 1Author : name of client.
  • 2Office : office.
  • 3Comment: comment.
  • 4Date Time : date.
  • 5Image : image of client.
  • 6Active : enable/disable.


Configure

  • 1Show Image : show image or not.
  • 2Thumb Width : width of thumb.
  • 3Thumb Height: height of thumb.
  • 4All Items : number items will load.
  • 5Item Show : number items will show each time.
  • 6Show DateTime : enable/disable.
  • 7Show Office : enable/disable.
